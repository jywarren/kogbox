Documentation

Kogbox uses only PHP, JavaScript, and plain HTML/XHTML for the time being... be patient! We intend to include other languages such as Ruby, Python, and more.

Kogbox functions:

include_snippet($user_id,$method_name [,$version])

Use this function to include other snippets' code in your own. We encourage users to write functions or classes so that their code may be reused by others.

To tie yours to a specific version (so that changes don't break your snippet), specify a version number. Otherwise it defaults to the most recent version.

get_snippet_info($user_id,$method_name [,$version])

Call this function to get a key/value array of information about the requested snippet... IF that snippet is public. Available information includes "version" (version #), "description", "total_use" (a tally of how many times that version has been run), "license_id", "ancestor_id", and more.

See this example for all the available fields, and sample code.
